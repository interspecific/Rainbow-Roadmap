Rainbow Roadmap

An Interactive Map and Resource Hub of LGBTQ, Climate, Environmental, Health, and Economic Data

This tool can be used to help make informed decisions when traveling and relocating in times of political hostility and climate change in the US.

Try:

  • Toggling layers on and off, zooming to different scales, and clicking on geographic features to see the underlying attribute information.
  • Clicking the "Show Layer List and Legend" button on the top right. Expand the layer groups to see data layers for that group. Use the "eye" icon to toggle visibility of layers. If a layer is greyed out and not showing up on the map, zoom in or out to make it visible.
  • Clicking on a feature to see the information associated with that particular feature. This will show you a table of data that has been aggregated about this location or feature.
  • Submitting a new safe location to the database using the button on the bottom left. These locations will NOT be public until they are reviewed by me. Please report any issues using the contact info in the header.
